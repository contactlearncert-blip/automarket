# Firebase Studio

This is a NextJS starter in Firebase Studio.

To get started, take a look at src/app/page.tsx.

## Déploiement sur Vercel

Vercel est une excellente plateforme pour déployer des applications Next.js. Voici les étapes recommandées pour déployer votre projet :

### 1. Pousser votre code sur un dépôt Git

Assurez-vous que votre code est sur un fournisseur Git comme GitHub, GitLab, ou Bitbucket. C'est nécessaire pour que Vercel puisse accéder à votre code.

### 2. Créer un compte Vercel et importer le projet

1.  Rendez-vous sur [vercel.com](https://vercel.com) et créez un compte. Il est plus simple de s'inscrire avec votre compte GitHub (ou autre fournisseur Git).
2.  Depuis votre tableau de bord Vercel, cliquez sur **"Add New..."** puis **"Project"**.
3.  Importez le dépôt Git que vous avez créé à l'étape précédente.

### 3. Configurer et Déployer

1.  Vercel détectera automatiquement que votre projet est une application Next.js et pré-configurera les bons paramètres de build.
2.  Vous n'avez normalement pas besoin de changer les paramètres par défaut.
3.  Cliquez sur le bouton **"Deploy"**.

Vercel va maintenant construire et déployer votre application. Une fois terminé, vous obtiendrez une URL publique pour accéder à votre site. C'est aussi simple que ça !
