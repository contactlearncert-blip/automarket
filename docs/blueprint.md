# **App Name**: AutoHub

## Core Features:

- Vehicle Listing: Display a comprehensive list of available vehicles with relevant details.
- Advanced Search: Enable users to search for vehicles based on make, model, year, price, and other criteria.
- Vehicle Details Page: Provide a detailed page for each vehicle, including specifications, photos, and seller information.
- Contact Seller: Allow potential buyers to contact the seller directly through the platform.
- AI-Powered Recommendations: Recommend vehicles to users based on their search history and preferences, using an AI tool to refine suggestions.

## Style Guidelines:

- Primary color: Deep blue (#1A237E) to evoke trust and reliability.
- Background color: Light gray (#EEEEEE), nearly white, for a clean, modern backdrop.
- Accent color: Light blue (#64B5F6) to highlight interactive elements and calls to action.
- Body and headline font: 'Inter', a sans-serif font providing a modern and clean reading experience.
- Use modern, minimalist icons to represent vehicle features and categories.
- Maintain a clean and organized layout, prioritizing easy navigation and visual clarity.
- Incorporate subtle animations for page transitions and loading states to enhance user experience.
